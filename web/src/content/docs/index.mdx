## What is byox?

byox is a single-screen TUI + engine that drives the **official open-source
course testers** from [CodeCrafters](https://codecrafters.io) — the exact same
checks the real platform runs — entirely on your machine, with stage-by-stage
progress tracking. No account, no submission queue.

## Start in three commands

```sh
git clone https://github.com/madhank93/build-your-own-x && cd build-your-own-x
mise install
just setup
```

See [Getting started](/getting-started/) for the full setup.
